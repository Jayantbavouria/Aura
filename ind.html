1. Meeting Setup

  • User creates an Agent (Persona/Instructions).
  • User schedules a Meeting linked to that Agent.
  • TRPC creates the call room in Stream.io immediately.

2. Live Interaction

  • User joins the call via /call/[id].
  • Stream webhook notifies your server.
  • Server connects the specific Agent's instructions to OpenAI Realtime API.
  • AI participates in the call as a user.

3. Post-Processing

  • Call ends -> Stream generates transcript.
  • Webhook triggers Inngest function.
  • Inngest parses transcript, maps speaker names, and uses GPT-4o to summarize.
  • Result stored in Neon DB.
